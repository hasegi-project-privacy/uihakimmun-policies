## 앱 정보

명(明) 이천(李梴)이 1575년에 엮은 종합 의서 「의학입문」을 원문(표점)과 한국어 번역·주해로 읽는 열람 앱.
계정·로그인 없이 개인정보를 직접 수집하지 않으며, 본문·번역은 기기 내에서 오프라인으로 동작한다.
무료 버전은 일부 편 열람 + Google AdMob 광고, 프리미엄(1회 구매)은 전체 편 열람 + 광고 제거로 제공한다.
